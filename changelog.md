# 更新日志

ℹ️建议在Windows应用商店下载和更新，支持自动更新，使用更加方便。

## v1.3.0 - 2026-03-25

- **改进更新检查和自动更新机制**，商店版本现在直接调用官方API进行应用内更新，更方便好用。
- **优化游戏/MOD文件夹选择功能**，在某些选错路径层级的情况可以自动纠错，减少让人摸不着头脑的报错。
- **优化Toast提示**，较长的Toast支持滚动查看，而非直接截断。
- **加入日志系统**，提供更详细的操作记录和错误信息，方便用户排查问题。
- **改进应用内公告**，支持Markdown渲染。

## v1.2.3 - 2026-03-24

- **上架 Microsoft Store**，提供更便捷的安装与更新方式。

## v1.2.2 - 2026-03-23

- **重构 UI 界面**，提供更简洁易用的视觉与交互体验。
- **重做刷新逻辑**，点击刷新即可一键检测本地未提交的改动及云端未拉取的更新。
- **新增昵称设置**，用于 MOD 上传至远程仓库时的作者署名（更新后首次启动将自动引导完成设置）。
- **优化日志显示**，全面支持多行文本展示与 Markdown 格式渲染。

## v1.2.1 - 2026-03-20

- **重构重定向模块**，改用内置的轻量级辅助 MOD 来实现重定向功能。

## v1.2.0 - 2026-03-20

- **新增存档重定向**，彻底替代此前存在缺陷的存档合并功能。
- **优化同步机制**，新增并完善了上传过程中的冲突检测与解决流程。
- **内置 MinGit 环境**，修复并完善对 Gitee 平台的支持。
- **修复异常提示**，解决了实际上传失败但错误提示为成功的问题。
