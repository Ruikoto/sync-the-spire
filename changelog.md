# 更新日志

ℹ️建议在Windows应用商店下载和更新，支持自动更新，使用更加方便。

## v2.1.0 - 2026-05-02

- **浅色模式**，新增浅色模式主题，可在设置中切换 跟随系统 / 浅色 / 深色
- **配置导入/导出**，支持导出和导入配置，方便在不同设备间同步、分享配置
- **超大文件保护**，自动检测工作区与未推送提交中的超大文件，推送前提示并提供"排除"或"撤销未推送提交"两种处理方式；自动识别 GitHub / Gitee 等平台的大小限制；支持在设置→高级设置中查看已排除文件列表并撤销排除
- **清理分支历史**，新增分支历史清理功能，用于彻底移除仓库中已存在的大文件
- STS2 修复重复 Manifest JSON 导致的 Mod 数量/差异/联机不一致问题，并在 Mod 管理中支持一键去重
- 修复文本框无法使用右键粘贴的问题
- 修复部分场景下凭证读取失败时仍尝试推送导致的"认证失败"错误
- 修复未安装系统 Git 时使用 ZIP 版无法 HTTPS 克隆的严重 Bug
- ZIP 版改为内置 MinGit + Git LFS，开箱即用，无需运行时下载
- 大幅精简安装包体积（自包含版省约 8 MB，MSIX 包省约 10 MB）
- Bug 修复、安全加固和性能优化

## v2.0.1 - 2026-04-17

- **多工作区**，全新UI，支持创建多工作区，可以同时管理多个游戏的 MOD 和存档
- **Mod管理**，新增Mod管理功能，可以更方便地安装、卸载、查看Mod详情
- **Mod差异比较**，在杀戮尖塔2上传、拉取Mod时，会自动比较并显示差异，更新前后的差异更直观
- **一键启动游戏**，新增启动游戏的按钮，一键启动游戏
- **其他游戏支持**，未来将新增对星露谷物语和Minecraft的支持，用户可以在创建工作区中选择游戏类型
- **多语言支持**，新增了对多种语言的支持，用户可以在设置中选择界面语言
- Bug修复和性能优化
