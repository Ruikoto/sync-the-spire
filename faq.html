常见问题 ​

没装过 Git 能用吗？ ​

能。软件会自动下载所需的 Git 组件，不需要你手动安装任何东西。

会影响游戏本体文件吗？ ​

不会。软件只操作 Mods 文件夹和存档文件夹，不修改游戏本体。关闭 Mod 开关即可恢复原版状态。

我的账号密码安全吗？ ​

认证信息使用 Windows 系统级加密存储，不上传到任何服务器，只有你本机当前用户能读取。

支持 Gitee 吗？ ​

支持。Gitee 仓库推荐用「免密」方式。
