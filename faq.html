常见问题 ​

没装过 Git 能用吗？ ​

能。软件会自动下载所需的 Git 组件，无需手动安装。

会影响游戏本体文件吗？ ​

不会。软件只操作 Mods 文件夹和存档文件夹，不修改游戏本体。关闭 Mod 开关即可恢复原版状态。

我的账号密码安全吗？ ​

认证信息使用 Windows 系统级加密（DPAPI）存储，不上传到任何服务器，仅本机当前用户可读取。

支持哪些 Git 平台？ ​

支持 Gitee、GitHub、GitLab 等主流平台。Gitee 推荐使用「免密」方式，GitHub 推荐使用「HTTPS + Token」方式。详见快速上手。

支持哪些游戏？ ​

目前完整支持杀戮尖塔 2。星露谷物语和 Minecraft 的支持正在开发中。你也可以选择「通用」类型来做基础的 Git 文件夹同步。

可以同时管理多个游戏吗？ ​

可以。v2.0 新增了多工作区功能，你可以为每个游戏创建独立的工作区，互不干扰。

界面语言可以切换吗？ ​

可以。点击仪表盘右上角的设置按钮，在设置面板中选择「语言」即可切换。目前支持中文和英文。

上传/拉取时进度条卡住怎么办？ ​

克隆或拉取操作取决于仓库大小和网络速度，较大的仓库首次同步可能需要较长时间。如果进度条长时间没有变化（超过 2 分钟），可以尝试关闭软件后重新打开。

Mod 差异预览显示什么？ ​

在上传或拉取 Mod 前，软件会自动比较本地和远程的 Mod 差异，显示新增、更新、移除的 Mod 列表，让你在确认后才执行操作。详见 Mod 管理。
