快速上手 ​

首次使用前请手动备份

备份你的存档和 Mod 文件夹。本软件经过初步测试，但多一道保险总没错。

选择 Git 平台 ​

Sync the Spire 需要一个远程 Git 仓库来存储和共享 Mod 配置。有以下几种选择：

平台说明
Gitee国内访问快，推荐国内用户首选。单仓库限 500MB，单文件限 50MB
GitHub全球通用，但国内部分地区访问受限
其他平台GitLab、Gitea 等理论上均可，需自行测试兼容性

创建远程仓库（以 Gitee 为例） ​

  1. 访问 Gitee 注册并登录。

  2. 点击右上角「+」→「新建仓库」

  1. 填写仓库名称（如 my-sts-mods），选择开源或私有，点击「创建」。
    • 开源仓库所有人可见，不限制协作人数
    • 私有仓库只有被邀请的人可见，免费账号最多 5 名协作者

  1. 创建成功后进入仓库主页，点击「HTTPS」，复制仓库地址。

使用 GitHub？

GitHub 整体流程类似，但认证方式需选择「HTTPS」，并在 GitHub 上创建 Personal Access Token 作为密码。具体步骤后续补充。

配置 Sync the Spire ​

  1. 打开 Sync the Spire，将刚才复制的仓库地址粘贴到输入框，填写你的昵称，认证方式选「免密」。

  2. 填写或自动配置好游戏路径和存档路径后，点击「保存并初始化」。完成时会弹出 Gitee 登录窗口，输入账号密码完成认证即可。

邀请队友 ​

推荐每位朋友各自注册 Gitee 账号，然后你在仓库里添加他们为协作者，各用各的账号，安全方便。

  1. 进入仓库「管理」→ 左侧「仓库成员管理」→「添加成员」。

  1. 权限选「开发者」，将邀请链接发给朋友，朋友接受后再把仓库地址一并发给他们。

  1. 朋友在 Sync the Spire 里填入同样的仓库地址，认证方式同样选「免密」，初始化完成后即可同步你的 Mod 配置。

认证方式怎么选？ ​

方式适用场景
免密公开仓库，或 Gitee 仓库（推送时系统会弹出登录窗口）
HTTPSGitHub 等平台的私有仓库，填用户名和 Token
SSH已配置 SSH 密钥的用户

游戏安装路径怎么找？ ​

Steam 中右键游戏 → 管理 → 浏览本地文件。

存档路径怎么找？ ​

  1. 启动游戏后按 ~ 键（波浪号键，通常在 Esc 下方）打开控制台
  2. 输入命令 open saves 并按回车
  3. 系统将直接打开存档所在的文件夹
